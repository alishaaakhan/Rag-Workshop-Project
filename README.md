# Rag-Workshop-Project
# 🩺 Medical FAQ Bot using RAG

A Streamlit-based AI application that analyzes uploaded medical reports using Retrieval-Augmented Generation (RAG).

## Features

- Upload medical report PDFs
- Ask questions from reports
- AI-generated explanations
- Uses Gemini + FAISS + LangChain

## Tech Stack

- Streamlit
- LangChain
- FAISS
- Gemini API
- HuggingFace Embeddings

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
